# task-tracker-cli

A simple CLI for tracking tasks.

Submission for https://roadmap.sh/projects/task-tracker

## Features

- Add a task
- Remove a task
- List all tasks
- Mark a task as done

## Setup

```bash
alias task-cli='python task-cli.py'
```

## Usage

```bash
task-cli add "Buy groceries"
task-cli remove 1
task-cli list
task-cli done 1 
```
